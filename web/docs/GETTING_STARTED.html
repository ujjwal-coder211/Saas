Getting Started with Aksh

Download ki zaroorat nahi. Sirf browser chahiye.

Step 1 — Free account

  1. Aksh Home par jao → Start Free
  2. Dashboard par email se signup (MVP)
  3. API key copy karo — ek baar dikhegi, save kar lo

Step 2 — Aksh Studio kholo

  1. Aksh Studio open karo
  2. Settings mein API key paste karo → Save
  3. New Project ya Upload zip (cloud account ke saath)

Step 3 — Sarva se baat karo

Chat panel mein likho:

Mujhe ek todo app banao React mein

Sarva code dega, files save hongi, thread memory on (SaaS DB ke saath).
