Sarva Memory

Har user ki alag chat threads — kisi aur user se mix nahi.

Studio / Chat

Thread sidebar se purani chat wapas kholo. Naya thread = nayi memory.

API

  • GET /saas/v1/threads
  • POST /saas/v1/threads
  • GET /saas/v1/threads/{id}/messages
  • POST /v1/chat with thread_id

Requires signup + Postgres (DATABASE_URL).
