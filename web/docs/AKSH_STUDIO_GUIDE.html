Sarva Studio Guide

Sarva Studio — browser IDE (Monaco editor + Sarva chat).

Panels

  • Explorer — project files, upload zip, new file
  • Editor — code edit, auto-save (cloud mode)
  • Chat / Agent — Sarva se baat; Agent tools use karta hai
  • Threads — alag chat history (SaaS account)

@file context

Open file checkbox on → active file Sarva ko bhejti hai.

.sarvarules

Project rules file — har request mein inject hoti hai.
